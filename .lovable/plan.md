

# Brown - Premium Bengali Fashion Website
## Phase 1: Navigation & Hero Section

### Overview
Building the foundation of Brown's premium e-commerce showcase, starting with the announcement bar, main navigation, and a cinematic hero section with polished carousel interactions.

---

### Component 1: Announcement Bar + Navigation

**Announcement Bar**
- Thin strip at the very top with Deep Espresso (#2E2319) background
- Left side: Smoothly scrolling marquee text — "PRE-ORDER • LAUNCHING EID 2026 • LIMITED PIECES • HANDCRAFTED BENGALI ELEGANCE"
- Right side: Static link — "JOIN THE BROWN CIRCLE [10% OFF FIRST ORDER]"
- Typography: Lora font, 11px, uppercase with letter-spacing

**Main Navigation**
- Clean Light Cream (#F8F5E9) header, 80px height
- Three-column layout:
  - **Left**: "Brown" logo in elegant Cormorant Garamond
  - **Center**: Horizontal menu links (Shop, Collection, Craftsmanship, About, Contact)
  - **Right**: Search, cart (with badge), and account icons
- Subtle hover effects on all interactive elements
- Responsive: Collapses to mobile hamburger menu on smaller screens

---

### Component 2: Hero Section with Carousel

**Layout**
- Full viewport height (100vh) with stunning lifestyle photography
- Dark gradient overlay for optimal text readability
- Left-aligned text content positioned elegantly

**Content Overlay**
- Eyebrow text: "CRAFTED FOR TIMELESS ELEGANCE"
- Bold headline: "Brown — Effortless Elegance, *Rooted in Heritage*" (with italic styling)
- Supporting body copy about small-batch panjabis
- Two CTA buttons: Primary filled + Secondary outline
- "Our Story →" link with hover animation

**Polished Carousel**
- 3-4 slides with high-quality stock photography placeholders
- Auto-advances every 5 seconds with smooth transitions
- Pauses on hover for better user experience
- Drag/swipe support for touch devices
- Elegant dot navigation in bottom-right corner
- Subtle crossfade or slide transitions

---

### Design System Setup
- Load Cormorant Garamond & Lora from Google Fonts
- Define brand colors as CSS custom properties
- Establish consistent spacing, typography scales, and hover patterns
- Mobile-responsive breakpoints baked in from the start

### What You'll See
A polished, premium first impression that captures Brown's sophisticated aesthetic — ready to review and refine before we move on to the product grid section.
